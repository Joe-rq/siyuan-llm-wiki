## 卡片结构（6页）

### P1 封面：钩子 + 标题
- 标题：AI帮我建了一座知识Wiki
- 副标题：思源笔记 + Claude = 自动知识库
- 视觉：一个大脑/笔记本图标 + "Not RAG, but Wiki" 标签

### P2 痛点共鸣
- 标题：收藏了100篇文章，记住几条？
- 内容：收藏夹是坟墓 / 笔记堆成山没有关联 / 知识打架了自己都不知道
- 视觉：堆积如山的文档 + 问号

### P3 核心方案
- 标题：丢一篇文章进去，8步全自动
- 内容：读源文档 → 创建摘要 → 提取实体 → 冲突检测 → 创建/更新实体页 → 更新主题 → 重写索引 → 追加日志
- 视觉：流水线/管线图

### P4 差异化亮点
- 标题：Obsidian做不到的事
- 内容：块级操作 — 改一个知识点，不用重写整篇 / 矛盾检测 — 新旧知识打架，AI自动发现 / 图谱查询 — 不是搜文本，是走关系链
- 视觉：对比表格或三列图标

### P5 四层防护
- 标题：AI写错了怎么办？四层防护兜底
- 内容：认知边界（每批≤4实体）→ 意图保护（三重校验）→ 执行可靠（写后验证）→ 持续优化（同类扫描+模式积累）
- 视觉：四层盾牌/嵌套方框

### P6 收尾CTA
- 标题：开源免费，思源用户直接用
- 内容：GitHub搜索 siyuan-llm-wiki / 安装后 /wiki 一条命令开始
- 视觉：GitHub logo + 思源logo + 扫码/链接指向

## 风格偏好
